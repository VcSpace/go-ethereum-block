## Deploy contract

```
solc --abi --bin -o ./ Storage.sol
abigen --bin=Storage.bin --abi=Storage.abi --pkg=storage --out=Storage.go
```

## interface

```
solc --abi -o ./ IERC20.sol
abigen --abi=IERC20.abi --pkg=IERC --out=IERC.go
```
